# Ascend PWA — simplest path for you

## Upload these files to your public GitHub repo
- index.html
- styles.css
- app.js
- service-worker.js
- manifest.webmanifest
- .nojekyll
- icons/icon-192.png
- icons/icon-512.png

## Then enable GitHub Pages
Settings → Pages → Deploy from a branch → main → /(root)
